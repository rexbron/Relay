Relay Help › Customization

Change the appearance

Color scheme

  1. Open Settings › Appearance.
  2. Choose from:
    • Light
    • Dark
    • System (follows your macOS appearance)

Each option shows a visual thumbnail preview so you can see the result before selecting it.

Message bubble style

Choose between two styles:

  • Grey — all message bubbles use a neutral grey background.
  • Colored — each sender gets a unique, deterministic color derived from their Matrix user ID, making it easier to follow multi-person conversations.

GIF animation

In Settings › Behavior, choose how animated GIFs play:

  • Always — GIFs animate continuously.
  • On Hover — GIFs are static until you hover over them.
  • Never — GIFs always show as a static image.

Tip: Relay respects your system accent color and accessibility settings, including increased contrast and reduced motion preferences.
