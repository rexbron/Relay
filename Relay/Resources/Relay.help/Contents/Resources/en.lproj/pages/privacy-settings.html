Privacy & Security

Privacy settings

Read receipts

In Settings > Behavior, toggle Send read receipts to control whether other users can see when you've read their messages. When enabled, Relay sends read receipts as you scroll through conversations.

Typing indicators

Toggle Send typing notifications to control whether other users see a typing indicator when you're composing a message.

URL previews

Toggle Show URL previews to control whether link previews are fetched and displayed inline in conversations.

Per-room overrides

In the room inspector's Behavior tab, you can override these settings on a per-room basis. Each setting shows whether it's using the global default or has been overridden. Click Restore All Defaults to reset all overrides.

Media preview scope

Choose whether media previews (images, videos) load automatically in all rooms or only in private (direct message) rooms.

Timeline events

Control whether membership events (joins, leaves, invites) and state events (room name changes, topic updates) are shown in the timeline.

Tip: Per-room behavior overrides let you be more private in public rooms while keeping convenience features in trusted private conversations.
