Change room settings

Open room settings

Click the room name capsule in the toolbar, or click the inspector toggle, to open the room inspector on the trailing edge.

General tab

View and edit the room's name, topic, avatar, and canonical alias. This tab also shows encryption status, visibility, room type, member count, pinned messages, and room ID. Authorized users can edit the name, topic, and avatar.

Room aliases

In the General tab, authorized users can add, remove, and set the primary alias. Alternative aliases can also be managed.

Members tab

Browse and search the member list. Each member shows a role badge (Admin, Moderator, or Member). Click a member to view their profile. From the profile, you can:

  • Send a direct message
  • Change their role
  • Kick, ban, or ignore them (permissions required)

Behavior tab

Override per-room settings for media previews, GIF animation, URL previews, membership events, and state events. Each setting shows whether it uses the default or has been overridden. Use Restore All Defaults to reset.

Notifications tab

Set the per-room notification mode:

  • All Messages
  • Mentions & Keywords Only
  • Mute

Choose Restore Default to use the global setting.

Security tab

View encryption status. Authorized users can toggle directory visibility, change join rules (Public, Invite Only, Knock), and set history visibility.

Permissions tab

Manage power levels for the following categories:

  • Room details — name, topic, avatar
  • Membership actions — invite, kick, ban
  • Messages — send, delete
  • Advanced settings

Roles are Admin, Moderator, Everyone, or a custom power level.

Note: Settings you can edit depend on your power level in the room. Greyed-out controls indicate you don't have permission to change that setting.
