Keyboard shortcuts

Relay supports standard macOS keyboard shortcuts for common actions.

Navigation

Shortcut Action
⌘K Quick Switch (jump to any room)
⌘G Search messages
⌘0 Bring Relay window to front

Rooms

Shortcut Action
⌘N Create a new room
⇧⌘N Create a new space
⌘J Join a room by address
⇧⌘D Open the room directory

Messages

Shortcut Action
↩ (Return) Send message
⌘E Edit your last message
⎋ (Escape) Cancel reply or edit

Windows

Shortcut Action
⌥⌘A Open the Activity Log
⌘S Export Activity Log (when Activity Log is focused)
⌘C Copy selected events as JSON (when Activity Log is focused)

Tip: Many standard macOS shortcuts also work in Relay, including ⌘, (Settings), ⌘W (close window), and ⌘Q (quit).
