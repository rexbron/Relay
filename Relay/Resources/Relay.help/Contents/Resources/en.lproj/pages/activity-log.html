Use the Activity Log

Open the Activity Log

Choose Window > Activity Log or press ⌥⌘A. The Activity Log shows a real-time scrollable table of diagnostic events from Relay's services.

Filter events

Use the filter controls to narrow the log:

  • Category: filter by event category (e.g., Call, Sync, Room)
  • Severity: filter by level (Debug, Info, Warning, Error)
  • Room: filter events related to a specific room
  • Free text: search event summaries and details

Inspect an event

Click an event to view its full details in the inspector panel, including timestamp, category, severity, source, summary, detail, room ID, and metadata.

Export the log

Press ⌘S in the Activity Log window to export the currently filtered events as a JSON file (default name: relay-activity-log.json). This is the preferred format for bug reports.

Copy events

Select events and press ⌘C to copy them as JSON to the clipboard.

Auto-scroll

The Activity Log auto-scrolls to show new events. Toggle auto-scroll off to freeze the view while investigating.

Tip: When reporting a bug, filter the Activity Log to the relevant category before exporting. This keeps the file focused and easier for maintainers to analyze.
