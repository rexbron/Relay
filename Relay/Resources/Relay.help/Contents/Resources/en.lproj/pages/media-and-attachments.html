Share media and files

Attach files

Click the attachment button in the compose bar or use the file picker. You can stage multiple files before sending.

Drag and drop

Drag files from Finder onto the compose bar or timeline. A drop target overlay appears to confirm the drop zone.

Paste images

Paste images from the clipboard directly into the compose bar with ⌘V.

Send GIFs

Click the # button in the compose bar to open the GIF picker (powered by Giphy). Search for a GIF and click it to send.

Captions

Add optional caption text to each attachment before sending.

Media previews

Images and videos display inline in the timeline. Audio files show a player control. Animated GIFs can be set to animate:

  • Always
  • On hover
  • Never

Control media visibility

In Settings › Behavior, you can control whether media previews load automatically, and whether they appear in all rooms or only private rooms.

Tip: You can also share files from other apps using the system Share menu. Shared files appear staged in Relay's compose bar for review before sending.
