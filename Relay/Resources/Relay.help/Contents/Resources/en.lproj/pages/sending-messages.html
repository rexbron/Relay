Send and edit messages

Send a message

Type in the compose bar at the bottom of the conversation and press Return to send. The compose bar supports multiple lines of text.

Format with Markdown

Relay supports Markdown formatting in messages:

  • **bold** for bold
  • *italic* for italic
  • `code` for inline code
  • [links](url) for hyperlinks
  • Triple backticks (```) for code blocks

Reply to a message

  1. Right-click a message and choose Reply, or swipe right on a message bubble.
  2. A reply preview appears in the compose bar.
  3. Type your reply and press Return to send.

Press ⎋ (Escape) to cancel the reply.

Edit a message

  1. Right-click your own message and choose Edit, or press ⌘E to edit your last message.
  2. The compose bar shows an Editing Message banner.
  3. Make your changes and press Return to save.

Press ⎋ (Escape) to cancel the edit.

Delete a message

  1. Right-click a message and choose Delete.
  2. A confirmation dialog appears before the message is removed.

You can delete your own messages. Room moderators can delete messages from other users.

Drafts

Unsent text, reply context, and staged attachments are automatically saved when you switch rooms and restored when you return.

Tip: Press ⎋ (Escape) to cancel a reply or edit and return to normal composing.
