Set up notifications

Default notification levels

Open Settings › Notifications. Set default notification behavior separately for direct messages and group rooms.

Per-room notification mode

In the room inspector's Notifications tab, override the default for individual rooms:

  • All Messages — notify on every new message.
  • Mentions & Keywords Only — notify only when you're mentioned or a keyword matches.
  • Mute — suppress all notifications from this room.

Mention notifications

Toggle notifications for when you're directly mentioned (@you) and when the whole room is notified (@room).

Keyword notifications

Add custom keywords that trigger notifications when they appear in messages. Keywords appear as tags in Settings › Notifications.

Call and invite notifications

Toggle notification alerts for incoming calls and room invitations.

Dock badge

Relay shows an unread count on its Dock icon. The badge respects per-room notification modes, so muted rooms don't contribute to the count.

System permissions

If macOS notification permissions are denied, Relay shows a warning in Settings › Notifications with a link to System Settings › Notifications.

Note: If you have custom notification rules from another Matrix client that conflict with Relay's settings, Relay can detect and offer to repair the inconsistency.
